# Student Grade Calculator

A simple Python-based console application for managing student grades. It allows adding student data, calculating totals, averages, and grades, displaying results, finding the topper, and saving data to CSV.


## Features
- **Add Students**: Input name, roll number, and marks for 5 subjects (Math, Physics, Chemistry, English, Computer Science).
- **Calculate Grades**: Automatically computes total, average, and letter grade (A-F) based on average.
- **View All Students**: Displays a table of all entered students with their details.
- **Find Topper**: Identifies the student with the highest average.
- **Save to CSV**: Exports student data to `students.csv` for easy record-keeping.
- **Validation**: Ensures marks are between 0-100.
- **Unit Tests**: Includes tests for the grading function using Python's built-in assert.
